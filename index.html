Cloud sync
Sync your manually-added activities, planned workouts, targets, races, and saved routes to the cloud so they stay consistent across phone, iPad, and computer.

One-time setup (~5 min):
  1. Sign up at supabase.com (free, GitHub login works)
  2. Click New Project. Name it anything, pick any region close to you, set a database password (you won't need it later).
  3. Wait ~1 min for the project to provision.
  4. Open the project's SQL Editor (left sidebar). Paste the SQL block below and click Run.
  5. Open Project Settings → API. Copy the Project URL and the anon public key.
  6. Paste both below, pick any User ID (e.g. your name), and Save. Repeat the paste step on each device.
SQL to paste in step 4 (click to expand) 
create table dashboard_data (
  user_id text primary key,
  data jsonb not null,
  updated_at timestamptz default now()
);

alter table dashboard_data enable row level security;

create policy "Allow all for anon"
  on dashboard_data for all
  using (true) with check (true);
